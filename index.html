• If you have any questions or bug reports, please contact me via DM: Here
• IceLobby is a free and fully customizable lobby-plugin for your spigot minecraft server.
• For working, you have to download the plugin and put it in the plugins folder of you spigot server.
• Feel free for using other plugins for the features this plugin does not support. Example: Tablist, Ranks, ...
COMMANDS x PERMISSIONS
Permissions and Commands

- /fly | icelobby.commands.fly
- /clearchat [cc] | icelobby.commands.clearchat
- /spawn set | icelobby.commands.setspawn
- /teleportall [tpall] | icelobby.commands.teleportall
- /globalmute [gmute] | icelobby.commands.globalmute
- /warp | icelobby.warp.manage

other permissions:
- gloabalmute bypass | icelobby.chat.bypass
- hotbaritems dropping | icelobby.items.drop
- hotbaritems moving | icelobby.items.move
SCREENSHOTS CONFIGS
There are 3 configs which can be modified for the best user experience.
The normal config.yml is for usual settings for example the default gamemode.
In the messages.yml you can edit nearly all the messages this plugin uses.
The navigator and the shop use the items.yml. You can edit every item with its usage.
